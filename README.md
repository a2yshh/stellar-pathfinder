# Stellar Pathfinder 🚀
### AI Search Algorithm Visualizer
 
An interactive space-themed web app that visualizes how AI search algorithms navigate a star map to find the optimal route between planets — with an adversarial enemy ship that tries to block your path.
 
---
 
## Algorithms Implemented
| Algorithm | Strategy | Optimal | Data Structure |
|---|---|---|---|
| BFS | Explore level by level | Yes (unweighted) | Queue |
| DFS | Dive deep first | No | Stack |
| UCS | Cheapest cost first | Yes | Priority Queue |
| DLS | DFS with depth cap | No | Stack |
| A* | Cost + heuristic | Yes | Priority Queue |
 
---
 
## Features
- Live step-by-step traversal animation across a 13-node star map
- Dropdown to select algorithm, source star, and destination star
- Adjustable visualization speed
- Enemy ship toggle — Minimax interceptor that blocks your optimal path
- Right panel explains the selected algorithm with complexity badges
- Mission log showing every node visited, cost, and frontier state
- Hover tooltip showing node type, edge count, and heuristic value (A*)
